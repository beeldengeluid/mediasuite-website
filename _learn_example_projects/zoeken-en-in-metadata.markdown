In this workshop, participants learn how to search in the Media Suite and its metadata. It includes a step-by-step guide, as well as reflection questions that can be answered individually or in a class-room setting. \
\
This workshop is meant to be given in a plenary setting and can be included into existing curricula. It exists of a handout document and a lecturers' manual, both in Dutch. \
\
Here, you can find both the tutorial and the lecturers' manual, including points for in-class discussion:

* [CLARIAH Media Suite Workshop: Zoeken en/in Metadata](https://zenodo.org/records/18712838)

This workshop was created as part of CLARIAH-PLUS: WP5: Media Studies (NWO).
