In this workshop, participants will learn how to search in the Automatic Speech Recognition transcripts in the Media Suite, how to refine this search and how to navigate through the actual transcripts. \
\
This workshop is meant to be given in a plenary setting and can be included into existing curricula. It exists of a handout document and a lecturers' manual, both in Dutch. The workshop is based on the tutorial:[ Searching and analysing the Sound and Vision Radio Collection using Automatic Speech Recognition](https://mediasuite.clariah.nl/learn/subject-tutorials/tutorial-searching-and-analysing-the-sound-and-vision-radio-collection-using-automatic-speech-recognition) (English).

Here, you can find both the tutorial and the lecturers' manual, including points for in-class discussion:

* [CLARIAH Media Suite Workshop: Automatic Speech Recognition](https://zenodo.org/records/18712971)

This workshop was created as part of CLARIAH-PLUS: WP5: Media Studies (NWO).
